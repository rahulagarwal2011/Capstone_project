# Results Tracking

Track all experimental results here with full provenance.

## Format
Each result entry must include:
- **Claim:** Which thesis claim (C1–C10) this supports
- **Date:** When the experiment was run
- **Commit:** Git SHA
- **Seed:** Random seed used
- **Dataset:** Name + SHA-256 hash
- **Hardware:** CPU/GPU, RAM, node count
- **Result:** The measured value ± CI
- **Status:** PROVEN / PARTIAL / OPEN / FALSIFIED
